# HelloWorld ツールプラグイン仕様

## 概要
HelloWorld プラグインは Dify のプラグイン開発チュートリアルで紹介されているサンプルです。ユーザーが指定した名前と挨拶言語に基づき、エージェントやワークフローから呼び出すと簡単な挨拶文を返します。

## ツール一覧
### `sayHello`
| パラメータ | 型 | 必須 | 説明 |
| ---------- | -- | ---- | ---- |
| `name` | string | 任意 | 挨拶する相手の名前。未指定時は `"World"` を利用します。 |
| `language` | string | 任意 | `"ja"` もしくは `"en"` を指定します。未指定時は日本語 (`"ja"`) で応答します。 |

### 応答
```json
{
  "message": "こんにちは、Dify!",
  "raw": {
    "language": "ja",
    "name": "Dify"
  }
}
```

## エラーハンドリング
- 未対応の `language` が指定された場合は `ValidationError` を送出します。
- 期待していない入力型が渡された場合も `ValidationError` を送出します。

## ログ
- 実行開始時に入力パラメータを `debug` ログで出力します。
- 応答生成後、最終的なメッセージを `info` ログで記録します。

## テスト方針
- `ja` と `en` の正常系をユニットテストでカバーします。
- 未知の言語が指定された場合のバリデーションエラーを確認します。
