## Dify Plugins

このプロジェクトは Dify の自作プラグイン集をまとめたものです
基本的なプラグインの開発は公式ガイドのプラグイン開発者ガイドラインに従います

https://docs.dify.ai/plugin-dev-ja/0312-contributor-covenant-code-of-conduct

## 開発フロー

codex web を利用する前提のレポジトリです
以下の開発フローに従うようにします

1. 対象プラグインの選定
  - 開発修正するプラグインを決めてフォーカスして開発します
  - 開発中は他のプラグインの修正は行いません
  - プラグインごとに src のサブディレクトリに開発成果物を管理します
2. 機能仕様はプラグインのサブディレクトリごとに仕様書を markdown に記載します
3. この機能仕様書が正しく作られていることを人が確認した上で実装をします
4. 機能追加は、まず機能仕様書を更新してから実装を始めます
5. 実装後は開発用のプラグインを CI/CD で公開してテストを行う。ただし、テストコードは事前に作成してCIで実行する
